\chapter{Дүгнэлт}
\label{Conclusion}
\newpage
Цахим дуудлага худалдааны системийг хөгжүүлж, худалдагч болон худалдан авагч нарт хэрэглэхэд хялбар, найдвартай, хүртээмжтэй орчин бүрдүүлснээр худалдан авагч, борлуулагчдыг холбож, аюулгүй, ил тод дуудлага худалдааны үйл ажиллагааг дэмжсэн.
\subsection{Манай системийн давуу талууд}
    \begin{itemize}
    \item Real-time data ашиглаж хэрэглэгчдэд үнийн өөрчлөлт болон цагийн мэдээлэлийг цаг алдалгүй хүргэх
    \item Unit test хийгдсэн 
    \item Integration test хийгдсэн
    \item Сүүлийн үеийн системүүдийг судалж өргөжүүлсэн
\end{itemize}
\subsection{Ирээдүйд хийж болох хөгжүүлэлтүүд}
    \begin{itemize}
    \item Худалдагч шууд дүрс ашиглаж дуудлага худалдааг явуулах
    \item Qpay төлбөрийн системийг бодит орчинд нэвтрүүлэх
    \item Блокчэйн суурьтай төлбөрийн систем
    \item Чат бот хэрэглэгчийн асуусан асуултад автомат хариулах
\end{itemize}
